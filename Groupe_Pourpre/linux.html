Linux

Il a été créé en 1991 par Linus Torvalds, un étudiant Finlandais d'Helsinki à partir de Minix (sorti en 1987), un système d'exploitation de type Unix minimaliste, dans le but de le faire évoluer et le rendre pleinement opérationnel.

Linux possède de nombreux avantages: Il est stable, multi-utilisateur, gratuit, et possède entre autres un très grand nombre de distributions ("versions") adaptées au bonheur de tout un chacun.

Nous utilisons la distribution d'UBUNTU lors de nos cours et travaux.

  • 1987 : création de Minix par Andrew Tanenbaum et ses élèves de la Vrije Universiteit d'Amsterdam.
  • 1991 : création de Linux par Linus Torvalds.
  • 1992 : Linux 0.96 avec interface graphique.
  • 1994 : sortie de Linux 1.0 le 12 mars.

Si vous aimez taper des commandes dans un terminal, en voici quelques unes à essayer !

cd : Naviguer dans l'arborescence système

cd /home/desktop/

mkdir : Créer un dossier.

mkdir /home/desktop/monNouveauDossier

touch/gedit/nano : Editer un fichier texte.

touch /home/desktop/monDossier/monFichier

echo : Remplacer le contenu d'un fichier texte

echo /home/desktop/monDossier/monFichier "Hello World"

echo >> : Rajouter du contenu à un fichier texte

echo >> /home/desktop/monDossier/monFichier "Ligne 2"

cat : Afficher le contenu d'un fichier texte

cat /home/monDossier/monFichier

mv : Déplacer ou renommer des fichiers ou répertoires.

mv /home/desktop/monDossier/monFichier /home/desktop/monNouveauDossier

cp : Copier Coller des fichiers ou répertoires.

cp /home/desktop/monNouveauDossier/monFichier /home/desktop/monDossier

ls : Afficher les dossiers/fichiers dans le répertoire actuel

ls /home/desktop/monNouveauDossier

ls -a : Afficher les dossier/fichiers cachés dans le répertoire actuel

ls -a /srv

sudo : S'octroyer les permissions super-utilisateur pour une commande (à placer au début de la commande)

sudo su : S'octroyer les permissions super-utilisateur

sudo mkdir srv/monSuperDossier

rm/rmdir: Supprimer un fichier/dossier

rm /home/desktop/monNouveauDossier/monFichier

rm -dir -rf: Supprimer un dossier qui n'est pas vide (ATTENTION)

rm -dir -rf /home/desktop/monNouveauDossier

man: Ouvrir le manuel d'utilisation du terminal

--help: Afficher les informations sur une commande

cd --help
