Git

Git, un outil de gestion des versions de projet décentralisé. C'est un logiciel open source qui facilite le coworking sur un projet en gardant les traces de chaque version et les modifications apportés par les auteurs

Pour commencer à utiliser Git, il faut l'installer. Si vous êtes sur une distribution Linux basé sur Debian, tel qu'Ubuntu, vous pouvez utiliser, à partir d'un terminal :

sudo apt-get install git

Une fois installé, vous pouvez commencer à interagir avec lui.

Créer un dépot git dans le workspace (à spécifier)

git init {name}

Créer un dépot git dans srv (à spécifier)

git init --bare {name}

Liste tous les nouveaux fichiers et les fichiers modifiés à commiter

git status

Montre l'historique des versions pour la branche courante

git log

Placer de nouveaux fichiers sous suivi de version

git add

Envoyer le fichier dans le dépôt git

git commit

Afficher les logs de l'auteur "Host"

git log --author=Host

Passer sur la branche "Master"

git checkout master

créer une nouvelle branche

git branch nomDeLaBranche

Lister les branches

git branch -a

Après un commit permet de nommer notre version en "v1"

git tag v1

Supprimer le tag "v1" mais conserve la version

git tag -d v1

Se connecter au dépôt git

git remote add origin /srv/passerelle.git

Afficher les connexions

git remote

Récupère tout l'historique du dépôt nommé et incorpore les modifications

git pull

Envoie tout les commits de la branche locale vers Github

git push [alias] [branche]

Télécharger un projet et tous son historique de versions

git clone [Lien]

Récupèrer la version "origin" sur la branche "master"

git fetch origin master

Fusionner la branche de dépôt dans la branche locale courante

git merge [nom de dépôt]/[branche]
