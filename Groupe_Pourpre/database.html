Database

Une base de données est une collection de données organisées de façon à être facilement accessibles, administrées et mises à jour. Les bases de données peuvent être classées par le type de contenu qu’elles renferment : bibliographique, full text, images ou des nombres.

Le terme Database est apparu en 1964, s'en suivirent de nombreux modèles de bases de données expérimentales et factuelles, d'abord utilisées par l'armée et les spécialistes du sujet. Apparues dans les années 1990, les bases de données objet-relationnel permettent d'organiser les données sous la forme de tables définies de façon formelle, à partir desquelles les données sont accessibles et assemblées sans avoir à réorganiser les tables de la base de données.

Nous utiliserons lors de nos cours le système de gestion de bases de données relationnelles (SGBDR) MySQL. Pour nous familiariser avec ce système, rien de mieux que la pratique !


Installer un émulateur de base de données en local, attention, lors de l'installation des différents paquets, il vous sera demandé d'entrer des mots de passes qui vous permetront par la suite d'accèder à vos bases de données.

Ouvrir un Terminal et entrer les commandes suivantes :

Installer les paquets Apache 2 nécéssaires à mysql

sudo apt-get install apache2

Installer les paquets mysql client

sudo apt-get install mysql-client

Installer les paquets mysql server

sudo apt-get install mysql-server

Installer les paquets de phpmyadmin

sudo apt-get install phpmyadmin

Une fois ces commandes exécutées, tous les paquets nécéssaires sont maintenant installés et vous pouvez commencer à intéragir avec la BDD

Tout d'abbord connectez-vous à mysql via la commande suivante :

mysql -u root -p

Il vous sera demandé un mot de passe, c'est celui que vous avez saisi lors de l'installation des paquets précédents, tapez-le sur votre clavier et appuyez sur Entrée. Vous êtes maintenant connectés à mysql et pouvez interagir avec vos bases de données, voici donc quelques commandes.

Créer une Base de Données

CREATE DATABASE dbName CHARACTER SET UTF8;

Supprimer une Base de Données

DROP DATABASE dbName;

Lister les Bases de Données

SHOW DATABASES;

Spécifier la Base de Données avec laquelle interagir

USE dbName;

Créer une Table dans une Base de Données

CREATE TABLE tableName;

Afficher la structure d'une table

DESCRIBE tableName;

Supprimer une Table

DROP TABLE tableName;

Renommer une Table

ALTER TABLE tableName RENAME AS newTableName;

Vider une Table

TRUNCATE tableName;
