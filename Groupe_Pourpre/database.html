Base de données

Une base de données (BDD) est une collection de données organisées de façon à être facilement accessibles, administrées et mises à jour. our accéder ou modifier les données, c’est nécessaire aussi d’utiliser un système qui va permettre de gérer cette base et un langage pour transmettre des instructions à la BDD.

Le système de gestion de base de données (SGBD) est un ensemble de logiciels permettant de manipuler les données d'une BDD, donnant une vue synthétique sur le contenu et permettant à plusieurs utilisateurs de manipuler la BDD simultanément. La majorité des SGBD sont basés sur le modèle client – serveur. Dans ce modèle, le client est le logiciel utilisé dans les requêtes des BDD et les serveurs, ordinateurs abritent le logiciel serveur, qui attendent les requêtes des clients et les répondent. Dans le cas de l’installation d’un SGBD basé sur ce modèle, vous installez le client et le serveur. Pour être capable se communiquer avec le logiciel client, il faut utiliser une langage de programmation que, dans le cas de MySQL est le SQL.
Les SGBD plus connus sont :

  • SQLite : libre et gratuit, n’utilise pas le schéma client - serveur et est limité en fonctionnalités ;
  • Oracle : il n’est pas libre, mais est complet et généralement utilisé par les grosses entreprises ;
  • Microsoft SQL Server : le SGBD de Microsoft, il n’est pas libre ;
  • PostgreSQL : libre et gratuit mais peu connu ;
  • MySQL : libre et gratuit, utilisé par le grand publique et pour professionnels.

Nous utiliserons lors de nos cours le système de gestion de bases de données relationnelles (SGBDR) MySQL. Pour nous familiariser avec ce système, rien de mieux que la pratique !


Installer un émulateur de base de données en local, attention, lors de l'installation des différents paquets, il vous sera demandé d'entrer des mots de passes qui vous permetront par la suite d'accèder à vos bases de données.

Ouvrir un Terminal et entrer les commandes suivantes :

Installer les paquets Apache 2 nécéssaires à mysql

sudo apt-get install apache2

Installer les paquets mysql client

sudo apt-get install mysql-client

Installer les paquets mysql server

sudo apt-get install mysql-server

Installer les paquets de phpmyadmin

sudo apt-get install phpmyadmin

Une fois ces commandes exécutées, tous les paquets nécéssaires sont maintenant installés et vous pouvez commencer à intéragir avec la BDD

Tout d'abbord connectez-vous à mysql via la commande suivante :

mysql -u root -p

Il vous sera demandé un mot de passe, c'est celui que vous avez saisi lors de l'installation des paquets précédents, tapez-le sur votre clavier et appuyez sur Entrée. Vous êtes maintenant connectés à mysql et pouvez interagir avec vos bases de données, voici donc quelques commandes.

Créer une Base de Données

CREATE DATABASE dbName CHARACTER SET UTF8;

Supprimer une Base de Données

DROP DATABASE dbName;

Lister les Bases de Données

SHOW DATABASES;

Spécifier la Base de Données avec laquelle interagir

USE dbName;

Créer une Table dans une Base de Données

CREATE TABLE tableName;

Afficher la structure d'une table

DESCRIBE tableName;

Supprimer une Table

DROP TABLE tableName;

Renommer une Table

ALTER TABLE tableName RENAME AS newTableName;

Vider une Table

TRUNCATE tableName;
