Git

Git est un moyen de versionner ses fichiers, il est très utilisé dans le domaine professionnel pour sa simplicité et sa practicité

Installer Git

sudo apt-get install git

Une fois installé, vous pouvez commencer à interagir avec lui.

Créer un dépot git dans le workspace (à spécifier)

git init {name}

Créer un dépot git dans srv (à spécifier)

git init --bare {name}

Vérifier l'état des fichiers

git status

Afficher les logs git

git log

Placer de nouveaux fichiers sous suivi de version

git add

Envoyer le fichier dans le dépôt git

git commit

Afficher les logs de l'auteur "Host"

git log --author=Host>

Passer sur la branche "Master"

git checkout master

Après un commit permet de nommer notre version en "v1"

git tag v1

Supprimer le tag "v1" mais conserve la version

git tag -d v1

Se connecter au dépôt git

git remote add origin /srv/passerelle.git

Afficher les connexions

git remote

Envoyer la version sur la branche "master"

git push origin master

Dupliquer un dépot .git

git clone /srv/passerelle.git

Récupèrer la version "origin" sur la branche "master"

git fetch origin master
